Primary quality gates:

- `cargo fmt --all --check`
- `cargo check --all-targets`
- `cargo clippy --all-targets --no-deps -- -D warnings`
- `cargo test --all-targets`

GitLab CI mirrors these checks through `.gitlab-ci.yml`.
