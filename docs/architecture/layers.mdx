Core boundary rules:

- `src/main.rs` starts the app and delegates to `app::run()`.
- `src/app/runtime/**` handles event handling and side-effect orchestration.
- `src/app/render/**` handles drawing and UI composition only.
- `src/app/state/**` stores state containers and state helpers.
- `src/app/orchestrator/**` owns spec/orchestration control and related formatting.
